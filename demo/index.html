How it Works

Open Browser is a non-custodial, multi-chain wallet SDK running entirely in your browser via WebAssembly.

01
WASM Sandbox

Rust compiled to WebAssembly (528KB). All cryptographic operations — key generation, signing, encryption — execute in an isolated, high-performance sandbox. No JavaScript touches your private keys.

02
Scrypt + AES-256-GCM

Your mnemonic is encrypted with scrypt KDF (N=65536, r=8, p=1) and AES-256-GCM with separated auth tag. Fully compatible with OWS CLI vault v2 format — same wallet opens in both.

03
BIP-39 + HD Derivation

One 12-word mnemonic derives addresses across 9 chains automatically. BIP-32 for secp256k1 (ETH, BTC, ATOM, TRX, XRP, FIL, Spark), SLIP-10 for ed25519 (SOL, SUI). Multi-account via index parameter.

04
EIP-712 Typed Signing

Full EIP-712 implementation — supports all atomic types (address, bool, string, bytes, uintN, intN), nested structs, arrays. Used for DeFi permits, token approvals, and X402 payment authorization.

05
X402 Payments

HTTP 402 Payment Required protocol for agentic commerce. When a server returns 402, the SDK automatically signs an EIP-3009 USDC TransferWithAuthorization and retries. AI agents pay for API access without human intervention.

06
API Keys + Policies

Create ows_key_ tokens for agents with HKDF-SHA256 encryption (separate from passphrase scrypt). Attach policies: allowed_chains, max_daily_spend, expires_at. All policies evaluated with AND logic before any key decryption.

07
Audit Trail

Every wallet operation is recorded in an append-only IndexedDB log — wallet creation, signing, key management, policy denials, X402 payments. Exportable as JSONL for compliance review.

08
AI Agent Runtime

Autonomous agents with natural language commands, scheduled/recurring tasks, conditional execution (balance thresholds, price alerts), task chaining, auto-retry with exponential backoff, and browser notifications.

09
LLM Integration

Connect OpenAI (GPT-4.1, o3), Anthropic (Claude Sonnet/Opus 4), Google (Gemini 2.5), DeepSeek, or Groq. API key stored locally only. Agent parses natural language into wallet actions via JSON responses.

10
Mainnet RPC

Live balance queries, USD pricing (CoinGecko, 30s cache), and transaction history across 9 chains using public mainnet endpoints. ETH transfers with RLP encoding, gas estimation, and broadcast.

11
Key Security

5-second TTL key cache with LRU eviction (max 32 entries). Keys zeroed via beforeunload/pagehide hooks. IndexedDB only — never localStorage. Scrypt downgrade attack protection validates KDF parameters on decrypt.

12
Vault Backup

Export entire vault (wallets, API keys, policies, audit log) as encrypted JSON. Restore on any device. All data remains encrypted — backup file is safe to store in cloud storage.

Quick Start

import init, { create_wallet, sign_message }
  from '@open-wallet-standard/browser';

await init();

const wallet = await create_wallet("my-wallet", "pass");
// wallet.accounts → 9 chain addresses

const sig = await sign_message(
  "my-wallet", "pass", "evm", "Hello"
);

Example: React Integration

import init, { load_wallet, sign_message }
  from '@open-wallet-standard/browser';
import { getBalance } from './js/rpc.js';

function useOWS() {
  const [wallet, setWallet] = useState(null);
  useEffect(() => { init(); }, []);

  const connect = async (name, pass) => {
    const w = await load_wallet(name, pass);
    setWallet({ name, pass, ...w });
  };

  const sign = (chain, msg) =>
    sign_message(wallet.name, wallet.pass, chain, msg);

  return { wallet, connect, sign };
}
